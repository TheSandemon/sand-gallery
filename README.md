# Sand-Gallery

A portfolio showcase for creative work, games, and apps.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS
- **Backend**: Firebase (Auth, Firestore, Storage)
- **Web3**: OnchainKit (wallet integration)
- **Animations**: Framer Motion

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

Deploy to Firebase Hosting:

```bash
firebase deploy
```

Or use the Firebase CLI:

```bash
firebase init hosting
firebase deploy --only hosting
```
