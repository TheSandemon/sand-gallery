# ClipForge — AI Content Repurposing Tool

## Project Overview
**What:** A micro SaaS that transforms long-form content (webinars, podcasts, videos) into platform-specific social media clips automatically.

**Why Now:** Marketing teams drown in content backlog. Hiring editors is expensive ($50-100/hr). AI can now reliably extract "golden moments" and format for LinkedIn, Twitter, TikTok, Instagram.

**Target:** B2B content marketers at growth-stage startups (1-3 person teams, $70K-$120K salary, $200-500/mo tools budget)

**Revenue Model:** Subscription ($29-$349/year)

## Market Validation
- Market size: $15.7B → $59.6B by 2030 (30% CAGR)
- Competitors: Opus Clip, Repurpose.io — gap is B2B-specific vertical
- Real pain: 2-4 hours per piece manual repurposing

## MVP Scope
**Phase 1 (Weeks 1-2):** Landing page + waitlist only
**Phase 2 (Weeks 3-4):** Waitlist + email capture
**Phase 3 (Weeks 5-6):** Beta with 10-20 users, OpenAI Whisper + GPT-4

## Tech Stack
- Next.js + Vercel (frontend + hosting)
- Supabase (storage, auth)
- Stripe (payments)
- OpenAI API (Whisper + GPT-4)
